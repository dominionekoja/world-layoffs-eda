# World Layoffs Data Analysis (EDA)

## Project Overview
This project explores global company layoffs across industries and countries using data cleaning and exploratory data analysis techniques.

## The goal is to identify:
- Layoff trends over time
- Most affected industries and countries
- Companies with the highest layoffs
- Impact of economic periods on layoffs

 ## Tools Used
- SQL (Data Cleaning)

## Key Insights
- Tech industry recorded the highest number of layoffs
- Layoffs peaked during global economic downturns
- US-based companies were most affected

## Project Structure
- Data: Raw and cleaned datasets  
- SQL: SQL scripts for data cleaning
